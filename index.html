Vim keybindings doc

Commands Implementation
vim "filename" opens a file in the vim editor
:q to quit vim without saving
:wq to save and quit the file
:w saving the file on the go
j moving the cursor down
k moving the cursor down
l moving the cursor right
h moving the cursor to the left
i pressing "i" results entering the "insert mode
dd deletes a line
G moves the cursor to the end of the file
gg moves the cursor to the top of the file
} moves the cursor down a block of code
{ moves the curser up a block of code
u undo
ctrl + r redo
yy copy a line of code
p paste a line of code
V visual mode activates
o adds a new line and enterns the insert mode
O adds a new line above and enters into the insert mode
w goes wone by one through the words
b goes backwards one by one through the words
:"number" takes the cursor to that line
0 beggining of a line
$ end of a line
W ignores punctuation
t! goto
